  <action>
Create a client component ('use client') that renders one guide section (either galaxies or objects). It receives:
- `type` — "galaxies" or "objects"
- `editions` — array of { slug, title, publishedAt, entryCount } for this type
- `sectionTitle` — display title ("Galaxias de la temporada" or "Objetos del mes")

Behavior:
1. **Edition dropdown** — `<select>` showing all editions for this type, formatted as title + date. Defaults to most recent (first in array since API returns sorted by publishedAt desc). When user selects a different edition, fetch its data from `/api/guides/public?slug={selectedSlug}`.

2. **PDF download button** — Sits next to the dropdown. Disabled with title="Disponible proximamente" (Phase 34 placeholder). Style it as a grayed-out button with a download icon or text "PDF".

3. **Filter controls** — Render filter pills/toggles for three dimensions. Each filter is a set of clickable pills that toggle on/off. When a pill is active, only objects matching that value show. Multiple active pills in same dimension = OR. Multiple dimensions = AND.
   - Equipment: Tel. inteligente, Equipo pequeno, Telescopio grande
   - Dificultad: Facil, Intermedio, Retante
   - Ubicacion: Ciudad, Suburbios, Oscuro

4. **Sort control** — Dropdown or toggle buttons for sort field:
   - Nombre (alphabetical by display name)
   - Hora optima (by optimalTime)
   - Dificultad (facil < intermedio < retante)
   - Magnitud (numeric, ascending = brightest first)

5. **Object list** — Render filtered+sorted entries using ObjectCard component. Use a responsive grid: 1 column on mobile, 2 on md, 3 on lg.

6. **Empty states**:
   - No editions at all: show "No hay guias publicadas de este tipo."
   - Filters produce no results: show "Ningun objeto coincide con los filtros seleccionados." with a "Limpiar filtros" button.

7. **Loading state** — Show a simple loading indicator while fetching a different edition.

State management: Use React useState for selectedSlug, filters (object with arrays for each dimension), sortField, guideData, loading. Use useEffect to fetch guide data when selectedSlug changes. On initial mount, auto-fetch the most recent edition.
